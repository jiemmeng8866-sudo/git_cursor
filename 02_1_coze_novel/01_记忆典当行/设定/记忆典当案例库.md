# 《记忆典当行》记忆典当案例库

## 案例库说明

本文档收录《记忆典当行》中所有重要的记忆典当案例，按卷次和章节排列，供创作参考。每个案例包含：客户背景、典当物、价值评估、交易过程、典当代价、后续发展、人性探讨要点。

---

## 卷一案例（第1-50章）

### 案例001：初恋消亡记

| 属性 | 内容 |
|------|------|
| **出现章节** | 第2-7章（以正文为准） |
| **客户姓名** | 沈云（设定旧稿曾用「林小满」，一律以正文为准） |
| **客户背景** | 女，二十出头；父母涉诉、医疗费承压；情感线与爷爷一辈记忆轮回交织 |
| **典当物** | 与「林逸辰」相关的初恋记忆（分两笔：赤色与高阶，金额与段数以正文为准） |
| **价值评估** | 随剧情揭示高于表面「红色」标价 |
| **典当所得** | 现金救命（五十万/两百万等以正文为准） |
| **典当代价** | 失去「如何去爱」的能力或情感空洞 |
| **交易过程** | 林渊用记忆之眼鉴定并完成契约；道德与规则拉扯 |
| **后续发展** | 离开 City、转账余韵与长线伏笔（无常气息等以正文为准） |
| **人性探讨** | 爱情是感觉还是能力？忘记痛苦的同时，是否也忘记了如何幸福？ |
| **经典台词** | "我只是想忘记他……"类——以成稿对白为准 |

### 案例002：死亡恐惧

| 属性 | 内容 |
|------|------|
| **出现章节** | 第9-10章、第15章 |
| **客户姓名** | 周德海 |
| **客户背景** | 男，68岁，退休教师，癌症晚期，被诊断只剩三个月生命 |
| **典当物** | "对死亡的恐惧" |
| **价值评估** | S级，本能恐惧，属于根源记忆范畴 |
| **典当所得** | 500万 + 平静面对死亡的心态 |
| **典当代价** | 失去对生命的敬畏，不再珍惜每一天 |
| **交易过程** | 林渊发现周德海恐惧的不是死亡本身，而是"被遗忘"。经过深入沟通，设计了一个特殊交易：典当的是"对被遗忘的恐惧"，而非生命本能。 |
| **后续发展** | 周德海平静走完最后时光，但在死前失去了"珍惜"的感觉。最后在睡梦中安详离世。 |
| **人性探讨** | 恐惧是诅咒还是祝福？死亡的恐惧是否也是生命的礼物？失去生命紧迫感的人生是解脱还是另一种遗憾？ |
| **经典台词** | "我以为我害怕的是死，后来才发现，我害怕的是被这个世界彻底遗忘。"——周德海 |

### 案例003：天才的陨落

| 属性 | 内容 |
|------|------|
| **出现章节** | 第11-12章 |
| **客户姓名** | 陈子墨 |
| **客户背景** | 男，35岁，著名钢琴家，10岁成名，被誉为"百年一遇的天才" |
| **典当物** | "对音乐的激情"（共计2000+段记忆） |
| **价值评估** | S级，灵魂根源记忆级别 |
| **典当所得** | 5000万 |
| **典当代价** | 再也无法感受音乐的美好，弹奏出来的音乐毫无灵魂 |
| **交易过程** | 陈子墨因长期高强度演奏产生严重的心理问题，看到钢琴就恶心。林渊警告他这是根源记忆，典当后将失去对音乐的一切感受。但陈子墨坚持，他以为只要技巧在就够了。 |
| **后续发展** | 陈子墨的技巧依然完美，但所有听过他演奏的人都说他"变了"。半年后被乐坛抛弃，因为他失去了让音乐打动人心的能力。 |
| **人性探讨** | 天赋与热情是否可以分离？失去激情的成就是真正的成就吗？技巧与灵魂，孰轻孰重？ |
| **经典台词** | "我以为我只是讨厌演奏，没想到我讨厌的是没有灵魂的演奏——而那正是我现在的全部。"——陈子墨（典当后） |

### 案例004：母爱无价

### 案例011：共犯的记忆

| 属性 | 内容 |
|------|------|
| **出现章节** | 第125章 |
| **客户姓名** | 张三、李四（共同申请） |
| **客户背景** | 搭档多年的窃贼，刚刚分赃不均反目成仇 |
| **典当物** | 一起偷盗的所有记忆（共有记忆） |
| **价值评估** | B级，共有记忆，处理复杂 |
| **典当代价** | 张三：失去所有赃款记忆；李四：失去所有犯罪技能 |
| **交易设计** | 这是首例"共有记忆"典当案例，必须双方同意。张三想典当赃款记忆独吞，李四想典当技能记忆撇清关系。但共有记忆无法分割，最终两人都只能选择全部典当或都不典当。 |
| **后续发展** | 最终两人都没有典当，因为在回忆中他们想起了最初合作时的情谊。多年的犯罪生涯中，他们是彼此唯一的依靠。 |
| **人性探讨** | 共犯是否可以分割记忆？背叛是否需要代价？利益面前，情谊是否还存在？ |

| **交易目的** | 因为记忆中藏着父亲牺牲的完整真相 |
| **后续发展** | 林渊选择典当，获得了完整的真相。但他也永远失去了对母亲面容的记忆，只能记住母亲的"感觉"。 |
| **人性探讨** | 为了真相需要付出什么代价？失去至亲记忆的人还是原来的自己吗？真相与记忆，孰轻孰重？ |

| **最终交易** | 林渊与冥王进行了"记忆法则"的最终对决 |
| **林渊的选择** | 林渊用自己最珍贵的记忆作为"筹码"，向冥王展示：正是这些痛苦又美好的记忆，才让人生有意义 |
| **交易结果** | 冥王最终放弃，因为他发现林渊的母亲曾经也想保护他——即使他后来变成了冥王 |
| **后续发展** | 冥王选择自我典当，封印了自己所有的记忆。他不再是冥王，也不再是林夜明，只是一个空白的存在。 |
| **人性探讨** | 进化是否需要抛弃情感？爱是否是超越一切的力量？救赎是否需要遗忘？ |
