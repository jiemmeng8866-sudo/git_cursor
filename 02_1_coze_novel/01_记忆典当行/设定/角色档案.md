# 一、主角阵营

## 林渊 - 记忆典当行主理人

### 基础档案

| 属性 | 内容 |
|------|------|
| **姓名** | 林渊 |
| **年龄** | 28岁（故事开始）→ 32岁（结局） |
| **性别** | 男 |
| **身份明面** | "记忆典当行"第四代传人，古董店老板 |
| **身份暗面** | 记忆宇宙"守夜人"第七席，规则执行者候选 |
| **外貌** | 清秀书卷气，圆框眼镜，白衬衫黑西裤 |
| **特殊标识** | 眼睛激活时呈银灰色 |
| **口头禅** | "每一笔交易，都有代价。" |

### 性格详解

**表层性格**：
- 冷静理性，说话点到即止
- 不苟言笑，但笑起来很温暖
- 做事深思熟虑，算无遗策

**深层性格**：
- 内心悲悯善良，只是藏得很深
- 对弱者有强烈的保护欲
- 因为从小失去父母，拒绝与任何人建立深厚羁绊

**成长弧线**：
- 第一阶段（1-50章）：从普通人到守夜人候补，学会用规则保护自己
- 第二阶段（51-150章）：从候补到正式守夜人，学会用力量保护他人
- 第三阶段（151-300章）：从守夜人到规则改革者，挑战旧秩序
- 第四阶段（301-400章）：从改革者到守护者，建立新秩序

### 能力档案

| 能力 | 等级 | 描述 | 副作用 |
|------|------|------|--------|
| 记忆之眼 | B级 | 可见记忆颜色、鉴定真伪、追踪记忆 | 眼睛变色、看到不该看的 |
| 典当行 | C级 | 空间折叠、记忆保管、规则解读 | 消耗精神力 |
| 记忆之血 | A级 | 记忆旅行、修复、净化、融合 | 消耗生命力、人格淡化 |
| 规则解读 | 候补 | 发现漏洞、合理利用、特殊申请 | 最终解释权在议会 |

### 关键关系

| 关系人 | 关系 | 情感浓度 | 备注 |
|--------|------|----------|------|
| 林守正 | 爷爷/养父 | 极高 | 失踪三年，结局被救出 |
| 林夜行 | 生父 | 极高 | 已牺牲，意识残留于墨羽 |
| 苏晚晴 | 生母 | 高 | 已故，意识碎片帮助过林渊 |
| 林夜明 | 叔叔/冥王 | 复杂 | 敌人，但有血缘羁绊 |
| 阿七 | 守护灵 | 极高 | 结局消散 |
| 墨羽 | 契约兽 | 高 | 体内有父亲意识碎片 |

## 阿七 - 记忆幽灵

### 基础档案

| 属性 | 内容 |
|------|------|
| **真实身份** | 林远山的记忆碎片凝聚体 |
| **外表年龄** | 12-13岁男孩 |
| **穿着** | 民国时期长衫 |
| **形态** | 半透明，飘浮状态 |
| **声音** | 稚嫩但带沧桑感 |

### 性格详解

**表层性格**：
- 毒舌傲娇，说话尖酸刻薄
- 嫌弃林渊笨手笨脚
- 经常说"你们林家人就是..."然后叹气

**深层性格**：
- 对林家有极深的执念
- 嘴硬心软，关键时刻比谁都紧张林渊
- 其实已经无法离开典当行，把林渊当成了自己的晚辈

### 能力详解

| 能力 | 描述 |
|------|------|
| 穿墙隐身 | 可以随意穿墙，能在特定区域隐身 |
| 记忆探查 | 可以探查进入典当行之人的记忆 |
| 预警危险 | 对恶意有强烈感知 |
| 记忆储藏 | 可以"保管"记忆，但不能单独执行交易 |

### 关键剧情

- **第5章**：首次现身，测试林渊资格
- **第26章**：透露典当行历史
- **第145章**：为加固封印牺牲，灵魂融入典当行
- **结局**：成为"记忆图书馆"管理员，不再消散

## 墨羽 - 契约黑鸦

### 基础档案

| 属性 | 内容 |
|------|------|
| **真实身份** | 林夜行的契约灵宠，体内有林夜行意识碎片 |
| **外表** | 通体漆黑乌鸦，金色眼睛，羽毛有墨色光泽 |
| **大小** | 普通乌鸦大小，可放大到翼展2米 |
| **声音** | 嘶哑低沉，偶尔会说人话 |

### 性格详解

**表层性格**：
- 高傲冷漠，只服从林家人
- 平时话很少，爱理不理
- 对林渊有强烈的保护欲但不表现出来

**深层性格**：
- 体内有林夜行的意识碎片，会在关键时刻以林夜行的口吻说话
- 一直在等待林渊成长为真正的守夜人
- 最终选择与林夜行意识一起消散，完成使命

### 关键剧情

- **第16章**：首次出现，带来守夜人议会消息
- **第73章**：掩护林渊逃脱，身受重伤
- **结局**：完成任务后消散，留下最后一句话："少主，老奴先走一步。"

| 属性 | 内容 |
|------|------|
| **真实身份** | 林渊的亲叔叔，林守正的次子 |
| **表面身份** | 慈善家，"明日制药"幕后控制人 |
| **年龄** | 55岁 |
| **外貌** | 儒雅中年男性，温和笑容下藏着疯狂 |
| **动机** | 认为人类情感是进化的阻碍 |
| **终极目标** | 打开记忆之门，让全人类失去情感记忆 |

- 温文尔雅，说话轻声细语
- 对弱者有"悲悯"——认为他们不该受苦
- 坚信自己是在"拯救"人类

**深层性格**：
- 童年被父母忽视，只有哥哥林夜行关爱他
- 一次背叛让他彻底黑化
- 其实内心深处仍然有对林家的眷恋

### 能力详解

| 能力 | 描述 |
|------|------|
| 记忆吞噬 | 可以永久吞噬记忆，使其彻底消失 |
| 记忆融合 | 可以将不同人的记忆融合 |
| 记忆之门 | 可以打开记忆之门的钥匙 |

### 关键剧情

- **第40章**：揭示真实身份
- **第138章**：叔侄对峙
- **第153章**：最终决战
- **第390章**：选择自我典当

### 经典台词

> "我不是在毁灭人类，我是在拯救他们。当他们不再被情感折磨，他们会感谢我的。"

> "渊儿，你不懂。你父亲太软弱了，他只知道守护，却不知道有时候'遗忘'才是最好的守护。"
