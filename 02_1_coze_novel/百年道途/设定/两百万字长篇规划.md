# 《百年道途》两百万字长篇规划

> 本规划基于《百年道途》小说设定大纲扩展而成，为200万字级别的超长篇小说提供系统化的创作蓝图。

---

## 第一部分：总纲规划

### 一、整体故事线

#### 核心主线：百年道途，三步登天

```
第一阶段：入门立命（1-100万字）
穿越异世 → 外门崛起 → 内门立足 → 封印觉醒 → 宗门扬名

第二阶段：开宗立派（100-170万字）
离开宗门 → 修仙界扬名 → 建立势力 → 揭开身世 → 对抗正魔

第三阶段：道成圆满（170-200万字）
掌控生死 → 地府复兴 → 飞升上界 → 超越仙界 → 圆满结局
```

#### 主线时间线

| 阶段 | 字数 | 时间跨度 | 核心事件 |
|------|------|----------|----------|
| 穿越觉醒 | 0-30万 | 前3年 | 命灯假死，灵根觉醒，葬经传承 |
| 外门崛起 | 30-60万 | 第3-8年 | 外门大比，秘境历练，封印二重开启 |
| 内门峥嵘 | 60-100万 | 第8-20年 | 内门争锋，地府传承，身世初现 |
| 扬名天下 | 100-140万 | 第20-40年 | 离开宗门，建立势力，正魔对立 |
| 生死博弈 | 140-170万 | 第40-70年 | 揭开身世，地府复兴，对抗上界 |
| 道成圆满 | 170-200万 | 第70-100年 | 掌控生死，飞升上界，超越极限 |

### 二、三大核心主线

#### 主线一：地府复兴

**核心目标**：恢复上古地府的荣光，重建生死秩序

**阶段目标**：
1. 觉醒传承，获得地府核心功法
2. 收集生死轮碎片，重铸神器
3. 统御鬼道势力，建立地府班底
4. 对抗毁灭地府的上古势力
5. 重开轮回，建立新的地府秩序

**关键节点**：
- 第30万字：获得《黄泉经》核心传承
- 第100万字：知晓地府毁灭真相
- 第150万字：生死轮碎片聚齐
- 第180万字：重建地府
- 第200万字：成为新一代地府主宰

#### 主线二：魂道开创

**核心目标**：以殡葬之术开创独特的魂道修炼体系

**阶段目标**：
1. 将现代殡葬知识与修仙结合
2. 开创独特的魂道功法
3. 建立魂修传承，收徒立派
4. 证明魂道的可行性与强大
5. 魂道成为修仙界公认的主流体系之一

**关键节点**：
- 第3章：获得《葬经注》，领悟三葬法基础
- 第10万字：开创"引魂术"基础功法
- 第50万字：总结"殡葬七术"理论体系
- 第100万字：建立"葬魂宗"雏形
- 第150万字：魂道功法完善，传承延续
- 第200万字：魂道大兴，与剑道、丹道并列

#### 主线三：百岁道途

**核心目标**：在百岁之躯的局限下，走出一条逆天改命之路

**阶段目标**：
1. 突破百岁寿元限制，延年益寿
2. 逐步恢复青春，返老还童
3. 突破凡人极限，成仙成圣
4. 超越天道压制，飞升更高层次
5. 最终超越生死，永恒不灭

**关键节点**：
- 第30万字：筑基成功，寿元延长至300年
- 第70万字：结丹成功，寿元延长至800年
- 第120万字：元婴成功，寿元延长至1500年
- 第160万字：化神成功，寿元延长至3000年
- 第200万字：大乘圆满，寿元万年以上

### 三、三大支线体系

#### 支线一：阴德体系

**核心设定**：通过行善积德积累阴德，获得天道认可和特殊力量

**支线内容**：
- 帮助亡魂安息，超度有缘之魂
- 消灭邪祟恶鬼，净化世间
- 治病救人，扶危济困
- 揭示真相，平反冤案
- 传承薪火，培养后辈

**阶段性任务**：
| 阶段 | 阴德积累 | 获得能力 |
|------|----------|----------|
| 初期 | 小德 | 修炼速度+10%，基础魂力强化 |
| 中期 | 中德 | 修炼速度+30%，魂力攻击增强 |
| 后期 | 大德 | 修炼速度+50%，天道护佑 |
| 圆满 | 圣德 | 完全掌控阴德之力，天道认可 |

#### 支线二：上古秘辛

**核心设定**：揭示上古地府毁灭的真相，串联整个世界观的隐藏历史

**秘辛内容**：
- 三万年前：天崩之战，地府陨落
- 一万年前：灵气衰退，传承断绝
- 千年前：轮回殿起源，阴谋初现
- 现代：各方势力角逐，生死平衡岌岌可危

**关键谜题**：
1. 地府为何毁灭？谁是幕后黑手？
2. 生死轮碎片散落何处？能否重铸？
3. 冥河为何选择主角作为传承者？
4. 上界势力与地府有何关联？
5. 主角前世今生有何秘密？

#### 支线三：情感羁绊

**核心设定**：修仙路上与各色人物的情感纠葛

**情感线设计**：

**师姐苏晴线**（主线情感）：
- 起点：宗门内的温柔师姐，初见时一瞥（第5章）
- 发展：共同历练，互相扶持
- 高潮：生死相依，感情升温
- 结局：双修伴侣，仙道同行

**王胖子线**（友情线）：
- 起点：秘境相识，臭味相投
- 发展：历经生死，肝胆相照
- 高潮：为我挡刀，义薄云天
- 结局：永久追随，葬魂宗副宗主

**冥河线**（传承线）：
- 起点：封印中的意识
- 发展：逐步觉醒，传授功法
- 高潮：牺牲自我，成全传承
- 结局：精神永存，寄魂于器

### 三、避免战力崩坏机制

#### 境界严格设定

| 境界 | 战力倍数 | 突破难度 | 崩坏风险 |
|------|----------|----------|----------|
| 炼气 | 1倍 | 低 | 低 |
| 筑基 | 10倍 | 中 | 低 |
| 结丹 | 100倍 | 高 | 中 |
| 元婴 | 1000倍 | 很高 | 中 |
| 化神 | 10000倍 | 极高 | 高 |
| 大乘 | 100000倍 | 极难 | 低 |
| 飞升 | 1000000倍 | 极限 | 低 |

#### 崩坏防止措施

1. **越级战斗限制**
   - 越1个小境界：可能
   - 越1个大境界：极难
   - 越2个大境界：不可能

2. **势力制衡**
   - 每个境界都有相应势力
   - 不存在单一大乘无敌
   - 势力间相互制衡

3. **天地压制**
   - 修为越高，天道压制越强
   - 大乘期后难以在凡界生存
   - 必须飞升

4. **资源稀缺**
   - 高级资源极度稀缺
   - 突破需要机缘
   - 不能无限叠加

5. **寿命限制**
   - 每个境界都有寿命上限
   - 无法长期闭关等待突破
   - 必须抓住时机
