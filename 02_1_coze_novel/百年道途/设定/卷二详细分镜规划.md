# 《百年道途》卷二《魂动外门》详细分镜规划

## 卷二总体信息

- **卷名**：《魂动外门》
- **章节范围**：第21-50章（30章）
- **字数目标**：约10万字（每章3500字左右）
- **核心冲突**：林夜在外门站稳脚跟，李峰从幕后走向台前，魂道体系逐渐暴露，筑基之路
- **当前状态**：林夜炼气9层巅峰，封印一重觉醒初期，已掌握葬经第二重葬魂术

## 角色状态

### 林夜（主角）
- **年龄**：103岁
- **修为**：炼气9层巅峰 → 筑基期
- **魂道能力**：葬气（一重）→ 葬魂（二重，逐渐掌握）→ 封印二重觉醒前兆
- **人设特点**：话极少、不解释、不动怒、遇事直接动手
- **殡葬知识**：镇魂、度化、风水、引魂、聚阴、处理尸体

### 李峰（反派）
- **定位**：从幕后走向台前，噬魂丹事件未彻底结束
- **状态**：暂时收手但继续暗中操作，对林夜保持警惕
- **势力**：外门第二大势力，与多位长老有关系

### 苏晴（早期盟友）
- **定位**：外门弟子，对林夜好奇
- **作用**：可能成为早期盟友，首次合作任务

### 张远山（暗中观察者）
- **定位**：外门长老的亲信或特殊存在
- **作用**：暗中保护与观察林夜，逐渐偏向林夜

### 赵峰（被击败者）
- **定位**：被林夜击败，心结未解
- **状态**：噬魂丹残余影响，心结与后续选择

## 第一阶段：暗流涌动（第21-30章）

### 第21章：暗流

**章节名**：暗流

**核心事件**：
1. 赵峰禁闭结束，回到外门，精神状态异常，时常出现幻觉
2. 李峰暗中派人监视林夜，试探他的虚实
3. 林夜在弟子房中修炼葬魂术，逐渐掌握感知神魂的能力
4. 外门其他势力（三长老系、二长老系）开始注意到林夜这个百岁弟子

**本章埋的伏笔**：
- 赵峰的幻觉是噬魂丹后遗症，暗示李峰仍在进行噬魂丹实验
- 监视林夜的人是李峰的心腹，为后续冲突埋下伏笔
- 林夜感知神魂能力逐渐增强，为后续搜魂、制造幻境做铺垫

**人设体现点**：
- 林夜对监视者不作任何表示，正常生活修炼
- 遇到赵峰时，没有嘲讽或得意，只是平静擦肩而过
- 修炼时专注，不与他人交流

---

### 第22章：魂息

**章节名**：魂息

**核心事件**：
1. 林夜运用葬魂术，感知到外门弟子中有异常魂息波动
2. 跟踪发现，是几个低阶弟子被暗中下药（噬魂丹残渣）
3. 林夜出手解除弟子体内的异常魂息，没有暴露自己
4. 李峰发现实验对象失效，怀疑有人干扰

**本章埋的伏笔**：
- 异常魂息与噬魂丹有关，李峰的实验仍在继续
- 林夜救下的弟子中，有一人后续会成为信息来源
- 李峰开始更加警惕，但不知道具体是谁在干扰

**人设体现点**：
- 林夜救人时不解释，不动手不动口，直接解除异常魂息
- 弟子感谢时，林夜只是点头离开，不说话
- 行动隐秘，不留痕迹

---

### 第23章：试探

**章节名**：试探

**核心事件**：
1. 李峰派人与林夜"偶遇"，试探他的实力和底牌
2. 来者言语挑衅，试图激怒林夜
3. 林夜不接话，对方突然出手，林夜一招制敌
4. 对方认输，林夜收手，没有下杀手，也没有多解释

**本章埋的伏笔**：
- 李峰从手下口中得知林夜实力不弱，但底牌未完全暴露
- 林夜的招式简洁有效，李峰怀疑他修炼了特殊功法
- 周围弟子注意到林夜的实力，开始讨论

**人设体现点**：
- 林夜全程不说话，对方挑衅时无动于衷
- 出手时快准狠，一招制敌，不拖泥带水
- 制敌后立即收手，不落井下石，转身就走

### 第24章：任务

**章节名**：任务

**核心事件**：
1. 外门任务堂发布特殊任务：清理荒山墓园的阴煞之气
2. 其他弟子认为任务报酬不高且危险，无人接取
3. 苏晴出现，对林夜产生好奇，主动邀请林夜组队
4. 林夜同意组队，没有多问苏晴的目的

**本章埋的伏笔**：
- 墓园任务与殡葬知识直接相关，林夜将展示专业能力
- 苏晴对林夜的好奇，暗示她可能察觉到什么
- 荒山墓园可能有特殊存在（亡魂、邪修等）

**人设体现点**：
- 林夜接任务时只看报酬，不问细节
- 与苏晴组队时，没有热情，只是点头同意
- 出发前不与苏晴聊天，独自准备所需物品

---

### 第25章：墓园

**章节名**：墓园

**核心事件**：
1. 林夜和苏晴抵达荒山墓园，发现阴煞之气比预想的重
2. 林夜运用风水知识，找出阴煞之气的源头（一处被破坏的风水局）
3. 苏晴第一次见识林夜的殡葬专业能力，震惊
4. 两人开始修复风水局，引出被困的亡魂

**本章埋的伏笔**：
- 风水局被人为破坏，暗示有邪修在暗中操作
- 被困的亡魂中有特殊存在（可能是前代修士）
- 苏晴对林夜的看法开始改变

**人设体现点**：
- 林夜检查墓园时专注专业，不向苏晴解释任何原理
- 修复风水局时动作娴熟，仿佛演练过无数次
- 与苏晴的交流极少，仅用简单手势或单字回应

### 第26章：亡魂

**章节名**：亡魂

**核心事件**：
1. 修复风水局后，数十道亡魂出现，其中有强大的主魂

3. 林夜运用葬魂术与度化手段，将亡魂逐一引渡
4. 强大的主魂在度化过程中，向林夜传递了重要信息

**本章埋的伏笔**：
- 主魂传递的信息与李峰有关，可能是重要线索
- 度化过程中，林夜获得阴德反馈，为后续阴德体系做铺垫
- 苏晴看到度化场景，对林夜的身份产生更多疑问

**人设体现点**：
- 林夜度化亡魂时态度庄重，动作专业，不向苏晴解释
- 接收主魂信息时表情不变，内心却在整理线索
- 度化完成后，没有向苏晴透露任何信息

## 写作注意事项

### 节奏控制
- 每3章一个小高潮（如第23章试探、第26章亡魂、第28章埋伏）
- 每10章一个中高潮（第30章准备、第40章夺取、第50章余波）
- 节奏张弛有度，避免连续高潮

### 人设保持
- 林夜全程话极少，不解释，不动怒
- 遇事直接动手解决，不做多余动作
- 殡葬知识融入动作中，不解释原理
- 情绪藏在平静里，用细节表现

### 伏笔埋设
- 每章至少埋1-2个伏笔
- 伏笔不要直接揭示，用细节暗示
- 重要伏笔在后续章节中逐步回收

### 爽点设计
- 林夜的每一步成长都要有爽点
- 战斗场面快准狠，不拖泥带水
- 用实力说话，不靠嘴炮
- 反派被打脸时，林夜不嘲讽，让事实说话

### 殡葬知识运用
- 镇魂：引渡亡魂，净化阴煞之气
- 度化：超度亡魂，获得阴德反馈
- 风水：勘察地形，修复风水局
- 引魂：引导魂魄，制造幻境
- 聚阴：聚集阴气，强化魂道
- 处理尸体：专业手法，不解释原理
