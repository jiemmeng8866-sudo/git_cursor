# 飞书文档批量导出指南

本文档详细说明如何将飞书云文档中的小说章节批量导出到本地。

## 📋 准备工作

### 1. 创建飞书自建应用
1. 访问 [飞书开放平台](https://open.feishu.cn/app)
2. 点击"创建企业自建应用"
3. 填写应用名称（如"文档导出工具"），应用描述可选填
4. 创建完成后，进入应用详情页

### 2. 获取应用凭证
在应用详情页找到：
- **App ID**：应用的唯一标识
- **App Secret**：应用密钥（点击"重置"可获取）

### 3. 配置应用权限
在"权限管理"页面，添加以下权限：
- `drive:drive`：云文档读取权限
- `drive:file`：文件操作权限

### 4. 发布应用
1. 在"版本管理与发布"页面，创建新版本
2. 申请发布，等待管理员审核（如您是管理员可自审）

## 🔧 获取文件夹Token

### 方法1：从URL提取
1. 在飞书中打开包含所有小说章节的**文件夹**
2. 复制浏览器地址栏的URL，格式如：
   ```
   https://my.feishu.cn/drive/folder/xxxxxxxxxx
   ```
3. **folder_token**就是URL最后的`xxxxxxxxxx`部分

### 方法2：使用开发者工具
如果URL不是上述格式，可以使用：
1. 在浏览器中按F12打开开发者工具
2. 切换到Network（网络）标签
3. 刷新文件夹页面
4. 查找API请求，从中提取folder_token

### 1. 修改配置文件
打开 `feishu_export.py`，修改以下配置：

```python

```

## 🔍 常见问题解决

### Q1: 权限不足错误
**现象**：`{"code":99991663,"msg":"No permission to access resource"}`
**解决**：
1. 确认应用已添加所有必要权限
2. 确认应用已发布
3. 确认登录账号有文档访问权限

### Q2: folder_token无效
**现象**：`{"code":99991663,"msg":"invalid folder_token"}`
**解决**：
1. 确认folder_token是否正确
2. 确认登录账号有该文件夹访问权限
3. 尝试使用文档token而非文件夹token

### Q3: 导出任务超时
**现象**：`导出超时: xxx.md`
**解决**：
1. 增加脚本中的`max_retries`值
2. 确认文档内容不是太大
3. 分段导出，每次少量文档

### Q4: 文件名乱码或非法字符
**解决**：脚本已自动处理文件名，如有特殊需求可修改`sanitize_filename`函数

## ⚡ 替代方案（如果API方案太复杂）

### 方案A：手动导出
1. 打开每个文档
2. 点击右上角"..." → "导出" → "Markdown"
3. 保存到本地目录

### 方案B：使用浏览器扩展
搜索浏览器扩展商店，查找"飞书文档批量导出"相关扩展

### 方案C：半自动化脚本
如果API权限申请困难，可使用模拟点击的自动化脚本（需Selenium）
