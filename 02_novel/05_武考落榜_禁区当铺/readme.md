# 《武考落榜？我继承了禁区当铺》

> 都市高武 × 当铺交易 × 人性抉择 × 悬疑爽文

## 书名备选

- 《武考落榜？我继承了禁区当铺》
- 《禁区当铺：典当天赋，按期赎命》
- 《高武当铺：客人典当的全是命》

## 一句话 Logline

落榜生继承一家只收「寿命、记忆、天赋」的当铺；每笔生意都是局，当期不还，客人会变成「货」。

## 核心机制（写作铁律）

1. **每笔交易三章内闭环**：典当—当期危机—赎当或爆雷。
2. **主角不能无限收**：收多了会被「当铺账本」反噬（业障/禁区标记）。
3. **客人即副本**：客人背景牵扯势力线，主线从碎片拼出父亲/前任掌柜之死。

## 第一卷目标

- 3～5 笔典型生意（天才、权贵、可怜人、恶人）。
- 引出「账本最后一页」空白之谜。
- 财团欲收购当铺资质。
