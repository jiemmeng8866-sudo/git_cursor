# 第一百一十七章：半魂

封印开启的那一刻，整个魔窟底层被暗紫色的光照成了白昼。是封印中六成魔力在一万年后第一次接触到主人时发出的极亮极烫的原始魔光。光从穹顶正中心那块凝固了万年的暗色太阳中往外迸射——每一道光束都是纯粹的九幽魔力，密度高到在空气中自己凝成了近乎液态的紫色晶丝。晶丝从穹顶往下垂——垂到夜姬头顶不到三尺处停住了。是半魂在认人。

夜姬没有闭眼。她把左手腕举过头顶——禁魔铃在铃壁接触到封印魔力的同秒炸开了一整圈从未有过的全频段铃音。是在和她一万年前自己设下的封印做最后的手续交接。交接的内容是：确认解除封印的人确实是本人。确认方式——是一段刻在铃舌内侧的太古魔宗契约原文。她一万年前写下来的。内容是："我自愿将自己六成魔力与本魂的右半部分剥离，形成独立的半魂封印。此封印只能由持有禁魔铃的本人解除。若本人已死——龙音替我开封印。若龙音已死——墨珩的第三号天工密钥自动触发。若墨珩也死了——那就让它封着。缺一不可。"

她还活着。龙音还活着。墨珩的石像还在第九十九层微微睁着眼。三个条件全满足。封印没有理由不解除。

锁从半魂外围开始一层层脱落——每一层脱落都伴随着一段禁魔铃音波被反向弹回。反弹的音波在铃壁上炸开的时候——铃壁上最外侧那两个学生名字最先碎掉。是字迹从铃壁上脱离，化成极细极亮的暗紫色光粉，在空气中悬了不到一瞬，然后灭了。光粉落在地面上印出了两个极浅极淡的名字倒影。倒影没有消失——名字没有了，但人的轮廓还在。夜姬低头看了一眼。没有停——继续开锁。

第二层锁。第三层锁。铃壁上每条名字都在承受一次从小到大的反冲音波。每碎一个名字她就在地面上留一道名字倒影。倒影越积越多——从两个变成了七排。整个训阵班的名字全在地上。然后是最内侧的名字——墨珩。龙音。她留着用的那两排名字在最大一次反冲波打到铃壁时才碎。碎的时候声音不一样——是极深极沉的一声闷响。像有人在很远的洞穴深处敲了一下一整面天工合金墙。闷响过后——铃壁上除了她自己的名字和万年前刻的原始魔宗正篆之外，所有后来加刻的名字全部消失了。

她自己的名字在铃舌正下方——那是禁魔铃出厂时墨珩替她刻的。"夜姬。"两个字。不会碎。除非铃碎了。这是墨珩给她造的铃——主人刻名在铸造时就被写进了天工合金的分子层。一万年不会掉。

铃壁上最后一道锁脱落。穹顶正中心的封印核心——那一整团六成魔力的暗色太阳——在所有封印剥离后变成了一团纯净的浮空液态魔力。颜色不再是暗紫——是被深渊瘴气污染了整整一万年后染上的一层极深极暗的黑。黑到透明度几乎为零。

"污染程度——比预计深至少四倍。"苏青黛把药鼎举到半魂下方。鼎心的青莲阵纹在接触污染魔力时自动切换到了全功率净化模式——不到半成的本源全部压进了阵纹中心。墨珩的净化阵台在中层持续运转——纯净的空气从头顶往下灌，把底层穹顶周围残余的深渊瘴气全部挡在外面。底层空间现在只剩下半魂本身携带的污染需要处理。她需要更多本源——不够。但墨珩的净化核心在感应到青莲本源不足时自动启动了应急预案。核心在中层阵台上把她刚才灌入的半成本源以一比一万的比例转换成天工编码的净化输出——是提高效率。天工坊的核心不懂修炼，但它能把每一丝青莲本源用在最精确的污染分子上。一万比一的效率意味着她需要输出的本源不到预计的万分之一。

半魂表面的污染层在净化阵台与药鼎的双重净化下从外往内层层剥离。剥离下来的污染碎片——每一片都是深渊瘴气在万年间与九幽魔力互相渗透后形成的混合杂质。杂质在空气中凝成极细的黑晶，然后被净化阵台吸入排水口——排入深渊裂缝本来的位置。墨珩的那句话在每一片杂质的归途中都被天工核心念了一遍。"从哪里来的回哪里去。"

污染清除。半魂露出了本来的颜色——极淡极清透的紫。和夜姬体内四成主魂的颜色完全一样——但更亮。因为这是六成。魂的颜色和清晨天刚亮时的远空一样——紫色底子上浮着一层极薄的暖光。一万年了——她第一次看到自己另一半灵魂的原色。

半魂在净化完毕的同秒开始往夜姬胸口的方向移动。是飞过去的。速度比所有人预计的都快——快到她还没把左手放下来，六成魔力已经穿过了禁魔铃的音障，穿过她的皮肤，穿过肋骨，穿过心脏外膜，在她心脏正中央停住。然后开始融。

融合的过程不到三息。但那就是最痛的三息。一万年前她自爆这六成魔力时是用禁魔铃把自己的灵魂硬撕成两半——撕的过程中痛到她快死掉的神经在最后关头被龙音一拳砸断。一万年后灵魂重新融合——是铸接。被撕开的灵魂断面上还残存着一万年前的旧伤——禁魔铃的音波在融合时需要用全功率把两道旧伤的边缘重新震活，然后在天工合金铃的频率导引下让它们重新长在一起。是愈合。但比撕裂更痛——因为撕裂时她以为自己不会活到缝合那一天。缝合时她知道会活。知道的痛比不知道的痛更实在。

她的左手在融合过程中把禁魔铃死死攥在拳心——铃舌被握到无法震动。她没有哭。但她把自己的嘴唇咬破了。破口处渗出来的血是紫色的——不是红色。九幽魔后的血从一万年前就是紫色的。龙音在音障外一直将右手放在离她肩很近的位置——没碰到。但这个位置很关键——万一她扛不住，一拳打在她左肩能把她的神经震麻而不伤到融合中的魂。一万年前在诸神战场上龙音帮她撕过魂——撕的位置也是左肩。今天在旁边等的也是同一只手。

夜姬的头发在融合完成时浮了起来——是十成九幽魔力在体内重新流通时产生的魔力余波把她全身的长发往上推了一把。头发从浮空慢慢落回肩头——每一根发丝上都残留着一层极淡极清透的紫光。她睁开了眼。左眼还是猩红——但红色更亮了，瞳孔深处多了一小片暗紫色的光斑。右眼的暗紫变得更沉——不再是那只睡了一万年后有点褪色的瞳孔。是太古魔后在全盛时期该有的颜色。她的赤足踩在地上——是一声极脆、极清、极安定的暗紫余韵。铃在说——主人完整了。铃是完整的铃。

她把左手从身侧抬起来。手指上——暖金符文环还在。世界树的契约在确认第三位女神碎片解锁完毕后自动在环内侧多写了一行新的字——和其他两位女神解锁时一样的通知格式。"九幽魔后——锁定碎片已清除。阵眼剩余：一。"

她低头看着自己的手。手上的指甲在刚才握拳时掐进了掌心——破皮了，但伤口已经在十成魔力的自愈下开始愈合。她把手翻过来。看着地面上那十几道名字倒影——每一道都在发光，倒影没有字了，光还在。

"铃上的名字还会再刻——是新的人。"她把右手翻过去，看着指甲上的暖金符文环。

龙音把搁在她左肩旁边的手收回去。没打出去。夜姬侧过头看了她一眼。"你欠我一拳。"

"留着。下次你掉魂的时候再打。"龙音把龙戟扛回肩上。

陆离把夜亮刀插在地上。阵眼在底层穹顶正中心——半魂离开后，封印的位置露出了被加固再加固后的弑神钉碎片。不大。比龙音那块小了不止一半。但碎片四周绕着一圈幽冥魔宗万年来加装的黑暗结界——是来加固封印的，不让别人从外面拆。现在封印从里面自己打开了。碎片外面加固了一层——没人拆。自己炸碎了自己的锁。碎片的固定架上刻着一行太古正篆：墨珩给夜姬的合同附件。附加条款：碎片需在半魂融合后一刻内拆除。拆除方式——九幽魔力与太初神纹同时输入。

夜姬把右手按在碎片上。她的魔力现在是十成——不需要省。陆离的左手按在她手背上。太初神纹的温度透过她的手背传到碎片。两种完全不同的能量——九幽的暗紫与神纹的暖金——在碎片内部碰在一起。是合作。万年前诸神战场上她们也是这样的。那年是剑祖和医圣——今天换成了魔后和守墓人。

碎片在双重输入下从内核往外碎裂。碎得比梵天和万道仙盟的两块都干脆——因为它面对的不仅有太初神纹的宇宙之心压制，还有整整十成完整的九幽魔力。九幽魔力是普天之下能正面接触弑神钉而不被共振反噬的极少数能量之一。魔后一万年前就已学会独自面对钉子——她只是没机会用全盛状态做。

碎片碎裂完毕。阵眼解锁。夜姬的半魂归位。共振链剩余——一。

（第一百一十七章完）
