# 第八十一章：苏醒

第三声铃响是陆离拨的。

他站在玄晶床前三尺处，把右手伸过去——用掌心的四十二度靠近了禁魔铃的铃舌。太初神纹的温度和铃舌之间的空气在那一寸间距里微微扭曲了一下。铃舌感应到神纹温度后自己倾斜，撞在铃壁上。

响了一声。极脆。和前两声不同——是被一个外人的体温逼响的。铃在表示确认——神纹宿主已到达。确认完毕。

夜姬睁开了眼。

左眼先开，右眼后开——间隔刚好一息。左眼猩红，右眼暗紫，异色双瞳在被封印一万年后第一次接触外界光线时瞳孔同时缩成竖线。在聚焦。她的瞳孔在没有任何预适应的情况下直接锁定了陆离的瞳孔。然后她眨了第一下眼。她在把一万年积压的魔气从泪腺膜上推开。推开之后她的眼神极其清醒。不像一个刚睡了一万年的人。像一个一直醒着、只是闭着眼等的人。

她的右手从左手腕上移开。左手腕上的禁魔铃在没有压制的第一时间自己响了一声——是整枚铃在九幽魔力重新流通时从内部炸开了一圈暗紫色的铃音涟漪。涟漪从穹顶扩到四壁，撞在墨珩刻的每道符咒上，撞回来时带回来一整面封印状态的读数。

所有人都在。龙音的三分之一龙魂在魂宫第三面墙里。苏青黛的青莲本源只剩半成不到。阿零脚踝上的龙纹纯度达到了祖龙令持有者的标准。庚辰体内的天工AI核心频率正在与穹顶的天工残片自发同步。洛清寒在门外——石门隔绝了她的弑神钉碎片共振。

她把这些信息在一息内全部收完。然后她悬浮起来了。赤足离床面半尺，身体还是侧躺的姿势，只是整个人从床上浮了起来。脚踝上的禁魔铃在浮空时清脆地响了一连串——是她睡觉时的姿势保持得太久，脚踝的关节在浮空后才开始恢复正常的活动范围。

她悬浮在与陆离视线平齐的高度。左眼猩红，右眼暗紫，眼角微翘。她的右手从身侧抬起来——把垂在脸旁的一缕头发往耳后捋了一下。动作很轻。然后她用左手食指指着陆离的胸口的太初神纹。

说了第一句话。"守墓人。"

她的声音比她看上去的年纪要沉——一种沉静。像一块被压在海底很久的石头捞上来后第一次被人敲了一下。余韵很长。

"你们陆家守了多久了？"

"十九年。"

她把左手食指收回去。然后她笑了——被一个答案逗到的笑。"一万年后的守墓人——只剩不到一年的寿命。你把宇宙之心扛在身上，连魂宫境五层都没突破。"她停了一下，右眼瞳孔微微扩了一圈。"谁让你进来的？"

"我自己。"

她歪了一下头。脖子往左偏了半寸——异色双瞳在歪头时从上到下打量了陆离一整遍。看的是身体上的伤口。六根肋骨的裂缝。左肩胛骨的骨裂。右手食中二指骨裂。腹部还没退完的毒印。太初神纹刚从冷却状态恢复。她的视线在每一个伤口上都停了一拍。

"你刚输过。"

不是问句。她从床上——从悬浮中——往前跨了一步。赤足踩在空中，每一步都踩出一圈暗紫色的涟漪。她走到陆离面前不到一尺处。她的身高比陆离低了将近一个头，但悬浮的高度刚好让她的眼与他的眼平齐。她把左手伸过去——没有碰到他的皮肤，只是用手掌悬在他胸口太初神纹的正上方半寸处。

神纹在她的掌心悬停时自主亮了一下——是识别。太初神纹认出了她。九幽魔后的九幽魔力——在万年前曾经和宇宙之心签过一份对等契约。是对等。夜姬在封印之前，宇宙之心曾经承诺过：当神纹宿主找上门时，她有权拒绝。但如果她自愿接受——契约自动成立。

"你知道太初神纹在我面前意味着什么吗？"她的声音很低。是在陈述。

陆离没有后退。"你是九幽魔后。神纹在你面前是钥匙。你能用它解开你自己的禁魔铃。然后你的完整战力不依赖任何人的本源——包括这颗宇宙之心。"

她停下了。歪着头看了他第二遍。看的是脑子。

"你说对了。禁魔铃是墨珩替我打的——为了防止我在沉睡期间魔力暴走炸掉整个西境。"她用右手食指敲了敲左手腕上的铃。"解开它需要太初神纹的完整激活状态——魂宫境九层的完整神纹输出。你只有魂宫境一层。"她把手指收回去。"所以你对我没用。"

她没有等陆离回答。她转过身——赤足在空中踩了半圈——然后往穹顶中心飘回去。她的背影很小。肩膀窄，腰背细，脚踝上禁魔铃在转身时响了一整串清脆的声音。和刚才铃的警告声不同——这一串是失望。纯粹的、不加掩饰的失望。她等了一万年，等来一个残血的神纹宿主。她还要再等。

苏青黛从地上把药鼎托了起来。是鼎身的青莲阵纹在夜姬转身时自动激活了两道净化纹。是药鼎在识别出九幽魔力纯度后本能地竖起了边际防御。夜姬停住了。她感觉到了青莲本源。

"青莲医圣。"她没有回头。"你也只剩半成本源。你的鼎上还有一道裂缝——虽然补过了，第七天会重新裂开。淬火的时候没用到极寒剑意，用的是国师府的替代方案。替代品撑不过七天。"

苏青黛没有说话。她把鼎托在掌心——没有收起来，也没有再激活阵纹。

然后夜姬把目光投向了阿零。阿零坐在通道口的石碑旁边，赤足搁在另一只脚上。脚踝的金色龙纹在暗紫魔气中格外醒目。夜姬的右眼——暗紫色那只——在看到龙纹时自动调整为竖瞳。是龙族与魔宗远古血脉协议中的自动识别程序。

"祖龙令。龙音的后人——不对。"她低头看了一眼阿零的赤足。"护魂后裔。龙音把她自己三分之一龙魂封在你身上了。你体内有祖妖血脉，不够纯，但够用。龙渊那小子签过约的——我看到你脚踝上的新龙鳞了。"

阿零没有站起来。她把脚踝往里收了收。她从夜姬身上感觉到的是一种更不好辨认的东西。是被迫的熟悉感。夜姬在认她——认的是她脚踝上的那枚龙纹。万年前龙音在封印之前来魔窟见过她一面。两人聊了整整一炷香。聊完之后夜姬把魔窟的石门关上了。龙音走的时候没有回头。

"她临走时跟我说了一句——"夜姬的声音忽然轻下来，是回忆的语气不需要大声。""你的人会来叫我的门。那时候我大约比你晚醒了很久。别吓着她。""她的左眼——那颗猩红色的——在回忆中暗了一瞬。"我等了一万年。是在等她说的那个人。"

她的头终于转过来，重新看向陆离。

"你不是她说的那个人。"

陆离把手甲上嵌着的浮游炮残片亮给她看。

夜姬低头看着那片残片。蜂窝结构。同一批次。同一架炮。她沉默了很久。是在算。她在算这个残片出现的时间。一万年前陆家第一代守庙人从这里借走了一整座浮游炮。三百七十年后还回来时残了。残片经过一万年演变——被阿零在军工城的废墟里捡到，被陆离嵌在手甲上。概率——她在心底用自己的九幽魔力推了一遍因果线。推过之后她没说出来。因果线没有间断。这不是巧合。

"你姓陆。"

"陆离。"

她把右手从身侧抬起来。是把手指拢成握一件东西的姿势。掌心向上，五指轻拢。九幽魔气在掌心中凝成了一把极小极暗的钥匙。是解锁禁魔铃的条件投影。投影中显示解锁需要：第一，魂宫境九层的完整太初神纹输出——现在不满足。第二，持有墨珩的天工残片足够启动铃的锻造回路——陆离的手甲里没有这个功能。第三——

第三条她没有投出来。她把投影捏碎了。

"第三条你不用知道。"

她把左手腕上的禁魔铃往里压了一下。铃没有响——是被她用手指制住了铃舌。然后她重新躺回了玄晶床。侧躺。右手压着左手腕。脚踝上的铃在身体重新接触床面时轻轻响了一下。她闭上了眼。

"你出去。等你魂宫境五层再回来——在此之前，不要碰我的门。"

她不说话了。睫毛安静地贴在眼睑上。呼吸频率还是六十息一下——是拒绝继续交谈。

陆离没有出去。他把怀里的铁链取出来——那条在医院储物间门上断开来的生锈铁链。锈迹里浸着一万年的九幽魔力残留。他把铁链放在玄晶床边上。

夜姬的睫毛动了一下。没有睁眼。

"你在港城医院碰过我的石碑。"她的声音比刚才更轻。是一个睡了一万年的人在被窝里不想起床的语气。"那条铁链是我锁门的——锁了几百年，铁锈里浸的都是我的魔力。你带着它走了这么远的路——就为了还给我？"

"是证明。"

"证明什么？"

"证明你不想被锁。你自己锁的门——用的是普通铁链。你知道铁链早晚会锈断。你等的是那个愿意扯断铁链走进来的人。"

夜姬睁开了右眼。暗紫色的那只。

然后她把右手从左手腕上移开，伸向铁链。她用手指在铁链的锈迹上抹了一下——铁锈里的九幽魔力回到主人手里时发出一声极轻微的叹息。是魔力本身在被释放时的回应。她把指尖上的铁锈搓掉。然后重新闭上眼。

"天亮之前。离开魔窟。"

是警告。她感应到了西境边界上的动静——幽冥魔宗的暗哨在沙坑外集结了。

（第八十一章完）
