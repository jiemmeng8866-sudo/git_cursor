## 一句话简介

极寒荒原的暴风雪中，众神祖庙的末代守庙人陆离，是个活不过20岁的"废柴病秧子"。当凌霄圣地的特使驾着百丈法相降临，要碾碎九尊女神石像铺路时——陆离咳着血，将手按在了居中的持剑女石像上。万年前被背叛的诸神，开始在他的血中苏醒。

## 核心卖点

1. **极致反差**：表面是将死之人的废柴守墓人，背后站着能毁天灭地的神明后宫团
2. **命运绑定**：强制的主仆/共生契约，高高在上的神明不得不倒贴主角
3. **征服感**：从相互利用到身心归属，攻略不同性格的远古女神

## 创作流程

1. **确定章节位置** → 查看 `02_Outline/` 下对应卷大纲，确认本章在框架中规定的核心事件
2. **执行写前检查** → ⚠️ 运行 `00_System/pre_write_checklist.md` 全部八项检查，输出自检报告
3. **查重+查伏笔** → 读取 `01_Bible/Event_Log.md` + `01_Bible/Anti_Repetition.md`，确认新章节不重复
4. **填写模板** → 打开 `00_System/chapter_prompt.md` 填入章节信息
5. **参考指导** → 翻阅 `00_System/writer_role.md` 回顾文风铁律与本书专属笔法
6. **生成正文** → 2000字/章，章末必须有Hook
7. **写后更新** → 更新 `01_Bible/Event_Log.md`（事件+伏笔+角色状态+系统功能）
