## 一句话简介
她是前世被鸩杀的开国女将，借尸还魂为病弱嫡女，手执盲杖再入棋局；他是手握重兵的摄政王，亦是前世唯一的宿敌与知己。这一世，他们从互相试探到背靠背厮杀，以天下为棋，共弈一场权谋长风。
## 核心设定
### 重生设定
- **前世**：燕长歌，开国第一女将兼军师，辅佐寒门未婚夫登基，功高震主被鸩杀，燕氏满门抄斩
- **今生**：借尸还魂为京城楚家常年卧病的长房嫡女楚惊微，伪装半盲，以羸弱之躯入局
### 核心卖点
**极致双强博弈（智性恋天花板）**
读者看的是"猎手与猎手的交锋"。男女主从互相试探、防备，到确认过眼神后背靠背杀敌。这种灵魂高度契合、利益深度绑定的关系，比单纯的甜宠更有张力和宿命感。
**降维打击的权力游戏**
女主前世是开国主帅，今生虽然身处后宅，但眼界是"天下局"。用打国战的脑子去打宅斗、打宫斗，这是一种"满级大佬新手村屠新手"的错位感。
**三大爽点**
- **破局爽**：反派布下看似无解的死局，女主毫发无伤破局，顺手赢走反派的底牌
- **掉马爽**：男主一次次试探，女主一次次化解。确认身份时那种隐秘的震动与压抑的保护欲
- **清算爽**：从第一卷孤立无援，到第七卷金銮殿铁证如山。看着仇人被一步步剥夺羽翼，信仰崩塌

## 创作工具指南

## 创作流程
1. **确定章节位置** → 查看 `timeline.md` 确认当前写到哪，伏笔回收情况
2. **填写模板** → 打开 `chapter_prompt.md` 填入章节信息
3. **参考指导** → 翻阅 `writer_role.md` 回顾文风要求（双强博弈、7:3比例）
4. **生成校验** → 对照质量评估标准检查输出，确认权谋逻辑自洽
