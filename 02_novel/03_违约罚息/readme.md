# 《高武：我的天赋是「违约罚息」》

> 都市高武 × 契约规则 × 智斗爽文 × 反杀压迫

## 书名备选

- 《高武：我的天赋是「违约罚息」》
- 《高武：违约越多，我利息越狠》
- 《契约高武：我专收霸王条款》

## 一句话 Logline

世人靠签约换资源，主角的天赋是「罚息」——谁对他毁约、压条款、玩阴的，就要按规则吐出利息；利息有时是力，有时是命，有时是地位。

## 核心机制（写作铁律）

1. **条款必须可见**：每章关键冲突要落在一两条可复述的规则上，避免空口开挂。
2. **罚息有类型**：气血债、天赋债、社会性债（曝光/降级/剥夺资格），轮换使用防腻。
3. **反噬边界**：主角主动制造不公契约会遭「罚息反噬」，限制滥强。

## 第一卷目标

- 立住「被踩→取证→立约/触约→罚息收割」循环。
- 校园或武馆场景切入，快速接财团/武盟合同副本。
- 埋：父母与某份「总协议」有关。
