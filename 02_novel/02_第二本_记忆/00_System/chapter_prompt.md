```
## 写作铁律

1. 白描手法：摒弃华丽辞藻和主观心理活动，只用精准的名词和动词
2. Show, Don't Tell：禁止在正文中直接科普世界观
3. 记忆剥离必须用通感写法（气味、温度、触感、声音）
4. 短句快节奏，段落不超过4行
5. 结尾断在悬念处（断章法：视角突变/反差惊悚/核心悬念）
6. 陆渊台词极简优雅，禁用感叹号和现代网络用语
7. 战斗必须具有物理打击感和血腥美学，禁止光波互轰
8. 每个出场角色必须有至少一个具体癖好或生理特征
9. 章末钩子不能重复已用过的断章模式
10. 琥珀的颜色/温度/气味必须与情感类型严格对应
```

## 防剧情重复协议（最高优先级指令）

> **核心原则**：每次写新章节/单元剧时，必须先读取 `01_Bible/Event_Log.md` 执行【查重检测】。绝不允许出现"换皮剧情"。

### 熔断机制

如果新剧情与已有剧情重合度超过 **30%**（客人身份雷同、核心情绪雷同、反转方式雷同），必须触发熔断：

```
[CRITICAL ERROR: 剧情雷同警告]
相似来源：[已有章节/单元]
雷同点：[具体说明]
建议：[提供变异方向]
```

### 禁令一：客户身份与诉求防重

| 已用模板（Vol 1） | 禁止再用的变体 |
|----------|---------------|
| 被霸凌的天才少女复仇（沈孤星，Ch1-4） | 任何形式的"底层弱者获得力量后暴力复仇霸凌者" |
| 过气演员渴望翻红（顾明川，Ch5-9） | 任何形式的"艺人/网红用容貌/才华换翻红" |
| 下城区黑诊所护士换弟弟的手（方晴，Ch12-13） | 任何形式的"姐姐/哥哥为救弟弟/妹妹牺牲器官/感官" |
| 绝症单亲父亲为女儿牺牲（周远山，Ch14-17） | 任何形式的"父母为绝症孩子典当自己" |
| 前天穹安全架构师吹哨复仇（老鬼，Ch18-22） | 任何形式的"前员工泄露公司机密复仇" |

**破解法**：
- 让客户身份反转——新客人本身是个加害者，来典当"罪恶感"以便继续作恶
- 让诉求变异——不是换能力，是换"被追杀的权利"、"被某人忘记"等反向需求
- 让结局出新——不是死/疯/入狱，可以是"赢了但主动放弃"、"发现代价是假的"

### 禁令二：付出代价与反噬防重

| 已用反噬类型（Vol 1） | 机制 |
|----------|------|
| 失去恐惧与共情（沈孤星） | 情感回路断裂：能计算眼泪流速但触发不了悲伤 |
| 失去爱的能力+吞枪自杀（顾明川） | 灰色纹路12%→40%→100%坍缩为灰白废琥珀 |
| 失去色觉（方晴） | 感官剥夺：世界变灰，但灰度辨识力提升 |
| 记忆全删+身体记忆残留+冻死（周远山） | 存在抹除：脑忘记但身体记得 |
| 永久不眠+神经坏死（老鬼） | 生理功能不经琥珀化直接凝固（透明睡眠之针） |

**第二卷必须使用全新反噬类型**：

| 可选类型 | 机制 | 示例方向 |
|------|------|------|
| A：感官剥夺（已用色觉） | 可换其他感官：听觉/味觉/触觉/痛觉 | 典当痛觉→被捅了也不知道，流血至死 |
| B：因果错乱（未用） | 典当恨意→爱上仇人，在清醒绝望中沉沦 | 复仇者典当对仇人的恨→反噬是保护仇人 |
| C：存在抹除（已用记忆） | 可换方向：不是被遗忘，是"所有记录"消失 | 典当"被世界记住的权利"→身份证件全失效 |
| D：镜像惩罚（未用） | 加害者典当愧疚→体验一遍受害者的全部痛苦 | 财阀高管典当"良心"→反噬是感同身受所有被他压榨的人 |

### 禁令三：战斗与高光防重

| 已用模板（Vol 1） | 禁止再用的变体 |
|----------|---------------|
| 苏离巨镰秒杀全部佣兵（Ch11） | 苏离一镰刀把XX全部砍翻 |
| 苏离1v7清道夫+1v3处刑人巨镰被锁用牙咬断喉咙（Ch21） | 苏离武器被锁→用身体部位击杀 |
| 陆渊门域展开注入恐惧碎片瘫痪敌人（Ch11, Ch21） | 陆渊站在门口展开领域秒杀全场 |
| 陆渊停止钟摆→天幕中断三秒（Ch22） | 陆渊改变城市级物理规则 |
| 沈孤星天台暴雨1v7+拳王反杀（Ch3） | 客人获得武力后在暴雨/高楼上反杀 |

**破解法**：
- 引入环境限制：在"渊"中战斗时物理攻击无效，必须用记忆琥珀作为规则武器
- 引入代价限制：客人用能力时有时间/次数限制，超限则反噬加速
- 利用智取而非暴力：本卷陆渊应更多使用微表情控制+心理侧写进行社交博弈

### 强制执行：AI 写作前置"四步自检"流程

每次下达"写下一章"指令时，AI 必须在输出正文**之前**，先输出以下自检报告：

```
【AI 创作前置自检】

1. 查重扫描：已写过的单元中是否有与本单元相似的元素（身份/情绪/反转方式）？
   [列出最相似的已有单元，说明差异点]

2. 逻辑连贯：上一章结尾的钩子是什么？本章开篇如何接续？
   [具体说明衔接方式]

3. 状态核对：陆渊当前技能库、渊化度百分比、已收集的琥珀颜色
   [列出当前状态，说明本章收益]

4. 新颖度打分（1-10分）：本单元核心设定的新颖度
   - ≥8分 → 通过，继续生成
   - 5-7分 → 警告，标注风险点，调整后继续
   - ≤4分 → 触发熔断，停止生成

--- 自检通过，正文开始 ---
```

## 质量评估

| 等级 | 标准 |
|------|------|
| A级 | 节奏紧凑，爽点密集，结尾悬念拉满，意象使用精准，通感描写到位，与已有章节零重复 |
| B级 | 故事推进正常，但缺乏亮点或通感描写不足，意向使用平淡 |
| C级 | 节奏拖沓，AI味重，出现心理直白描写或世界观科普，与已有章节疑似重复 |

## 绝对禁止

- ❌ 心理直白描写（"她感到……"、"他意识到……"）
- ❌ 世界观教科书式科普
- ❌ 总结性结尾（"这一夜，陆渊明白了一个道理……"）
- ❌ 现代网络流行语（卧槽、破防、绝绝子等）
- ❌ 感叹号连击（！！）
- ❌ 五颜六色的光波战斗
- ❌ 纸片人NPC（无特征、无癖好的路人）
- ❌ 与已完成五单元任何维度的重复（情绪/结局/反噬/场景/琥珀/签约模式/通感）
- ❌ "不是...是..."句式连续使用超过2次/段（Vol 1修复后标准）
- ❌ 楚歌出场标配"膝盖咔嚓+打火机按三次+风衣拖积水"三连（Vol 1修复后已拆分）
