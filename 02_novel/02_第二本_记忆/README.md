## 一句话简介

霖州永远在下雨。下城区最深雨巷的幽蓝木门后，有一家当铺。你可以典当任何记忆——恐惧、爱情、良知——换取你想要的一切。但当陆渊发现渊不是天灾，而是人造乌托邦逃避痛苦的产物，他必须在"封渊让人类重新承担痛苦"与"放任渊扩散"之间做出选择。

**全书主题**：不做神，做人。痛苦是自由意志的代价。

### 核心边界（防止混淆）

| | 01_Bible（参考文档） | 02_Outline（大纲文档） |
|---|---|---|
| **存什么** | 已发生的、不可变的事实 | 未发生的、可改的计划 |
| **更新时机** | 写完一章/一卷后 | 规划新弧段时 |
| **内容类型** | 角色当前状态、已发生事件、已使用设定 | 待完成弧段、待收束线索、角色终态目标 |
| **不能做的事** | 不能写"接下来计划XXX" | 不能写"ChX 发生了XXX"的流水账 |
| **权威性** | 事实权威——新章必须与之保持一致 | 方向权威——写作时参考，但可以调整 |

### 写新章时的使用顺序

1. **先执行** `01_Bible/Pre_Write_Checklist.md` → 五步检查（定位弧段→防重复→角色状态→设定→伏笔）
2. **再读** `02_Outline/当前卷_Outline.md` → 确认本章在弧段中的具体位置
3. **参考** `00_System/writer_role.md` → 回顾文风铁律（角色语调/氛围准则）
4. **写完后更新** `01_Bible/Event_Log.md` → 追加新条目（事件/设定/场景/伏笔/角色状态）
5. **角色状态有重大变化时** → 同步更新 `01_Bible/Character_Sheets.md`
6. **引入新设定时** → 同步更新 `01_Bible/World_Rules.md`
7. **卷末** → 更新 `01_Bible/Timeline.md`
