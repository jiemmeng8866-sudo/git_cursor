# 第81章：围城

霖州。天穹大厦。八十七层。凌晨五点零三分。

信号发射器通电了。瞬间——全功率——频率——每分钟四十九次——和第一代守门人的脊椎节数一致——和当铺门楣上的第零代手印一致——和陆渊右臂里门扇的共振频率一致——和钟阎掌心新骨的生长速度——一致——和阿加莎铜青色红线——在风中振动的波节——一致——和第七圈产下的四十九粒六角形卵——排列时的间距——一致。所有和渊有关的东西——都在同一个频率上——因为渊只有一个。一个渊——一个频率——被天穹——采集——放大——然后——通过八十七层的窗玻璃——往全城——广播。

第一波信号——从八十七层——到巷口——直线距离四公里——走了——不到零点零二秒。信号——撞在当铺门楣上——模仿。发射器——模拟了第零代手印的频率——模拟得——几无二致——差的一点——是温度。第零代的手印——是三十七度——信号——是冷的——没有温度——只有频率。但门——认不出温度差——只认频率。门楣——在收到模拟频率的瞬间——八层手印——同时——亮了——不是暖金——冷白——天穹的冷白——和神仙水吧台的日光灯管——同款。

门——开始锁。

所有的缝隙——同时——往中心挤。门扇——门框——门楣——门槛——每一块木头——都在收缩——紧——紧到——门缝——从三分之一——变回了一条线——然后——消失了——门——完全——密合了。但里面——还有一个人——方知远。方知远——被困在当铺里面——被关。门不让他出——因为门——在收到假频率后——把所有进入的人——都识别为——"守门人的身体"——要保护。保护——就是锁在体内——茧。当铺——在信号覆盖的瞬间——从建筑——变成了茧——茧——会把里面的所有人——裹住——不让外面的假频率——碰到——然后——茧——会下沉——沉入渊——沉入门的另一侧——沉入不夜城——在那里——假频率——够不到——假的——在渊里——会被渊自动识别——然后——吐出来——吐出来——代价——代价——是假频率的制造者——天穹——会被渊——反振——从八十七层——振碎——每一面玻璃——每一个终端——每一个头骨。

但方知远不要茧。他在当铺里——站在柜台前——柜台——七个凹槽——六个——有琥珀——第七个——空的——第八个——空的——第九个——在等。他低头——看着七个凹槽——然后——把右手——食指——那道极细的铜绿色线——按在第七个凹槽底部——写字——用指尖的铜离子——在铜制凹槽的底部——写了"挽"。字——在凹槽里——画——手指划过铜面——铜离子——从指腹——渗进凹槽——在凹槽表面——形成了一层极薄的——铜绿——铜绿——排列成——提手旁——加——免——"挽"——不是放手。

然后——第七个凹槽——亮了——暖金。暖金——来自内——钟歧封在凹槽里的——婴儿时期的陆渊——第一次哭——被方知远的一个"挽"字——重新触发了。温度——三十七度——婴儿的哭——是三十七度——和守门人掌心——一致。这个温度——钟歧——三十年前——在陆渊刚被取走记忆后——在凹槽里——存了一点——体温——陆渊——能记住的——第一口——不是奶——三十七度——是暖——是他还活着——的——确认。

然后——暖金——从第七个凹槽——往外——漫到第六——第五——第四——第三——第二——第一——七个凹槽——同时——亮了——暖金色——回应——回应门——回应茧——门——在暖金漫到第一凹槽的瞬间——停了收缩——犹豫。门——认出了自己的频率——真的——三十七度的——和外面天穹的冷白——差了一度————人在——和——机器在——的区别——门——认出来了——然后——门缝——重新——开了一条缝——四分之一——够一只手——伸进来——或者——伸出去。方知远——把右手——从那道缝里——伸出去——手指——在门楣上——那个悬着的第九个手印——还差不到一毫米就按上去的位置——用指尖的铜绿——画了一个字——"郑"——左边——关耳——右边——"尊"的一半——"郑"——姓。然后——"尊"没有写完——他不替她写——等她来——自己写。

巷口——凌晨五点零三分半。

陆渊到了。黑伞——没有撑——伞尖——在青砖地上——拖——留印——伞尖——在青砖上——划出了一条——极细的——暖金色线——从巷口——延伸到当铺门口——三米——线——弯的——弯成了——一个字的形状——在青砖上——"钟"——偏旁——只有一半——金——另一半——木——在第七号当铺的门楣上——隔着三百公里——在海底——在降温——但还亮——每七秒——和爬虫同步。陆渊——在青砖上——用伞尖——写了自己的偏旁——然后——停在当铺门口——看着门楣——看着九层手印——前八层——已经印实了——第九层——还在悬——还在等。他看着那扇开了一条缝的门——看着方知远伸出来的右手——食指上——铜绿——在凌晨的脉冲里——不是暖金——铜绿色——介于渊和现世之间的颜色——然后——他用自己的右手——握住了方知远的手指——碰——碰了一下——指尖——他的——暖金——方知远的——铜绿——在接触的瞬间——两种温度——在两种金属——之间——融合了——变成第三种温度——不冷——不热——常温——是室温——是巷口此时的空气温度——是霖州的温度——是人间的温度。

"门——不让我出去。"方知远的声音——从门缝里——传出来——被门吸收了一半——传出来的——只剩一半——另一半——被门——当成了第一次——来自活着的人——不是守门人——在门里的——声音——然后——它把那半句——存了——存在了柜台底下——第六代落地钟的钟摆里——当成了——门的第一句——记忆。

"不让你出——就暂时别出。你在里面——台上有七块琥珀——六块——是客人的代价——一块——是第七凹槽里的——暖金——是我第一次哭的温度。我祖父封存的——我还没用到——你帮我——先拿着——拿在手里——三十七度——它会告诉你——哪块琥珀——对应哪个凹槽——感觉——暖金——碰到猩红——会烫——碰到灰白——会冷——碰到七彩——会麻——碰到纯暖金——会共振——碰到透明——会静——碰到暗紫——会恶心——碰到暗金——会重。你感觉到了——就把对应的琥珀——从凹槽里——拿出来——拿在手里——不要放——等我进来。"

"你要怎么进来——门锁了——假频率——还在广播——天穹——不会停。"

陆渊没有回答。他把右手——从方知远指尖——移开——然后——把左手——按在门楣上——第九个手印的位置——悬——手心离手印——不到一毫米——暖金色的——从掌心——能看见——第九个手印——空——在等——但他没有按。手印——不是他的。门醒了——要的第九个手印——第九代。第八代——已经在门上了——在门扇里——在他的右臂——他整个人——就是一扇门——不需要再在手印里——加一层。

他把左手——从手印上移开——然后——把黑伞——撑开——伞面——骨白的——在巷口——对着天穹大厦——对着八十七层——对着那个还在广播假频率的信号发射器——挡了一下。挡视线。天穹的发射器——用光学瞄准——光学——需要直线——需要无遮挡——黑伞——伞面——骨白的——在光学波段——是反光的——不——吸光——骨白色——在幽蓝和暗红脉冲下——吸收所有波长的——百分之九十九——剩下的百分之一——被散射——散射成——没有频率的——灰——灰——在光学瞄准镜——一片——盖住了——整个巷口——整个当铺——整个青砖巷——发射器——看不到当铺了——就调不准频率——频率不准——假频率——就和真频率——差了一点点——一点点——门——就能分辨——分辨——就不锁——不锁——就能开——能开——就能进——能进——陆渊——就能进去——和苏离——和楚歌——三个人——一起——在当铺里——站——在假频率——被挡在外面的一刻——深呼吸一口——然后——等天穹——下一次——也是最后一次——物理攻击。

"苏离。楚歌。进来。"

陆渊推开了门。用右臂。右臂——门扇——和门楣上的旧门扇——在触及时——吻合——新门——嵌进旧门——旧门——认出了新门——门缝——从四分之一——变成了全开——消失了——门——通道——通道——从巷口——直接——通往柜台——通往凹槽——通往落地钟——通往二楼——通往苏离的房间——通往郑挽此刻——推开窗——跳下来——落地的——那一秒——她——在门口——站直——右手——在那个悬着的第九个手印上——放——放了一下——掌心——三十七度——然后——手印——印上去了——第九代守门人的助手——守——一个拿钥匙的人——不是铜牌——不是骨环——不是灰白点————牙。她缺了一颗牙——五岁换牙时——自己拔的——新牙顶旧牙——顶松了——然后自己拔——放在枕头下——第二天——没有硬币——没有牙仙——只有一张纸条——母亲写的——"自己拔的牙——自己留着。"她留了——十七岁——牙还在口袋里——现在——她把那颗牙——从口袋里——拿出来——用掌心——按在了第九个手印中央——还——还——还给自己——告诉门——"我不是守门人——但我有一颗乳牙——是我的——可以抵押——等守门人来了——再还我。"

然后她走进了当铺。

苏离跟着她进来——巨镰——横过门框——黑点——在进门时——闪了一次——暖金——门——在收频率——把刚才天穹打进来的冷白——都收了——然后——还给巨镰——转成账——门——在被假频率吵醒之后——学会了记账——每一份假频率——记在天穹名下——下一次——天穹有人推门——门——会先问——"你欠的——八个假频率——一个——值一根手指。""八毫"——门的新单位——一毫——等于一个守门人的手印——加一个假频率——门——自己发明了——自己——的——货币。

楚歌最后进来。右肩——暗纹——在进门时——被门楣——扫了一下——登记。门——在他的暗纹上——找到了渊的标记——"选中替代铁渣第四根手指"——门——把这个标记——转成了守门人的格式——加——楚歌——现在是——不是守门人——但被门——认可为——"编外——第一毫"。编外——不算代——但门——会保护他——像保护守门人一样——因为他是渊的——渊选的——门——听渊的——渊——听合约的——合约——记录——铁渣——欠第四根手指——楚歌——替铁渣——在肩上——长了第四根——新的——第四根——门——认——认了——就加——加了——他就不是刑警了——不是典当者——是门的一部分——不是被锁——是被接受。

门在他身后——关了。留缝——留了——刚好够——一个人的——眼睛——从外面——往里面——看到——柜台——看到——方知远站在柜台后——手里——捧着第七个凹槽里的——暖金——三十七度——正在——按他的指引——把琥珀——一个一个——拿出来——握在另一只手里——六块琥珀——六种温度——在掌心里——排队——排队——站在各自的凹槽上面——等——陆渊——进来——告诉它们——接下来——会发生什么。

陆渊站在柜台前。右手——暖金线——从掌心——延伸——现在——已经过了肘——快到肩——再往前——就是——门扇的核心——不——是——他自己的核心——门扇——他的身体——在变身——变回——原始——门——渊给他第三个字——"门"——种——他这一种——不是人——门——人形的门——门——不需要代价——门——只需要——钥匙。钥匙——九代——每一代——一把——他是第八把——第九把——在靳明舟手里——在铜室——还没出来——还没接受——但在门楣上——第九个手印——被郑挽的乳牙——暂时——定住了——能等——等到靳明舟——自己从铜室——走上七级骨阶——推开青砖巷深处的门——然后——顺着巷子——走到当铺——在门楣上——用自己的掌心——印上——印——守——是守门人的——第九代——然后——门上——所有层的手印——就会完整——完整——就可以——同时——关上——完成——完成之后——渊在霖州的入口——就会被这门——取代——取代——转移——从井口——从地下水——从废土——从死海——从渊眼——全部——转到这扇门——然后——这扇门——由九代守门人——共同——看——看管——每五十年——付十分之一——继续——继续——直到——渊自己——退回到——它不该被打开的那个地方——那个——人造乌托邦——排出第一铲废料——的地方。

然后——门外——巷口——天亮了。天幕——暗红色——在退——退回到——它三天前的面积——暗绿色——也从四分之三——往回缩——被收——被渊收回去——因为渊——在退——在第七圈开始孵化反语——在第零代自由——在钟阎接手井口——在陆渊变成门——在所有人——都在付十分之一——的时候——渊的水位——在降——天幕——在变——变向——第三种——不是暗红——暖金——门的光——门——醒了——发出的——温度——三十七度——把霖州——整个——罩住了——保温——保持——每一寸——地面——每一个人——皮肤——感官——在三十七度——正好——够——人——感觉——自己——还活着。

然后——天穹八十七层的玻璃——在暖金光照到的瞬间——裂了。裂——每一条裂缝——都是六角形——和第七圈产的卵——同一个形状——和温镇海铜室上方——针孔的符号——同一个形状——和钟阎掌心新骨——的晶型——同一个形状。六角形——渊的语言——是"还"——天穹——欠渊的——今天——开始——到期了。

不是还——不是用钱——用每一面玻璃——每一块电路板——每一个签名——每一个被取走的记忆——全部——在一个字里——在"C"——不——在"还"——在所有——在——暖金的——三十七度——的——门——醒来的——第一天。
