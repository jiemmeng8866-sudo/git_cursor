## 一句话简介

一把算盘从乡野破局，现代商战思维搅动乱世风云。当阶下囚变成执棋人，她与那位冷心冷情的权臣，算了一笔名为"天下与共"的账。

## 核心主题

- **小人物逆袭**：罪臣之女从阶下囚到掌控国家经济命脉的崛起之路
- **女性独立意识**：不依附、不恋爱脑，信奉"真金白银比人心更可靠"
- **现代思维降维打击**：将现代金融、物流管理、供应链思维应用于古代商战
- **势均力敌的爱情**：男女主从互相算计到性命交托的灵魂伴侣

## 创作工具指南

| 文档 | 定位 | 使用场景 |
|------|------|----------|
| `readme.md` | 项目总导航 | 新人上手、全局概览 |
| `settings/world.md` | 世界观词典 | 写到相关设定时查阅 |
| `settings/characters.md` | 人物档案库 | 角色出场时参考人设 |
| `settings/timeline.md` | 剧情进度表 | 确保不丢伏笔、节奏合理 |
| `prompts/writer_role.md` | 写作风格指南 | 每次写作前快速回顾 |
| `prompts/chapter_prompt.md` | 章节指令模板 | 写新章节时直接填入 |

## 创作流程

1. **确定章节位置** → 查看 `timeline.md` 确认当前写到哪，伏笔回收情况
2. **填写模板** → 打开 `chapter_prompt.md` 填入章节信息
3. **参考指导** → 翻阅 `writer_role.md` 回顾文风要求
4. **生成校验** → 对照质量评估标准检查输出，确认权谋逻辑自洽
