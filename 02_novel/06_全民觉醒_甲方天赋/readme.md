# 《全民觉醒：我觉醒的是「甲方天赋」》

> 都市高武 × 反套路觉醒 × 轻松热血 × 打脸爽文

## 书名备选

- 《全民觉醒：我觉醒的是「甲方天赋」》
- 《高武：我把天才变成乙方》
- 《觉醒甲方后，全服欠我违约金》

## 一句话 Logline

全民觉醒战斗天赋，主角觉醒的是「甲方」：在一定条件下把对手变成乙方——不履行约定就扣气血、扣天赋、扣装备使用权。

## 核心机制（写作铁律）

1. **立约要有前置**：如压制、见证、公开挑战、或对方先动手；避免无脑言出法随。
2. **违约金可视化**：读者要看到「扣了什么、当场效果」。
3. **喜剧与压迫平衡**：可玩梗，但主线要有生死压力（财团/禁区）。

## 第一卷目标

- 校园或训练营三连打脸，建立「甲方」名场面。
- 引出「天赋公证」与主角异常登记。
- 埋：甲方天赋历史上出过「暴君」，被联合封杀。
