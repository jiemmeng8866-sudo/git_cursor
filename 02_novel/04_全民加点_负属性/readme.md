# 《全民加点，我加的是「负属性」》

> 都市高武 × 面板加点 × 反套路成长 × 热血爽文

## 书名备选

- 《全民加点，我加的是「负属性」》
- 《高武：我把痛觉点满后无敌了》
- 《别人加力量，我加羞耻和失眠》

## 一句话 Logline

全民觉醒面板，唯独主角的「负属性」能换成禁忌收益：痛觉、羞耻、恐惧、睡眠需求……点得越狠，换到的隐藏属性越离谱。

## 核心机制（写作铁律）

1. **负属性→正收益闭环**：每条负属性必须对应可写清的兑换（例：痛觉↑ → 伤害判定阈值/应激爆发）。
2. **兑换有冷却或代价**：避免无脑叠；可用心智污染、社交崩盘、失控半章。
3. **公开处刑感**：前期被当疯子，中期一次名场面翻盘。

## 第一卷目标

- 三次「负加点」示范 + 一次大翻盘。
- 引出「面板来源」：禁区协议 or 实验体。
- 对手从同届天才升级到财团观察组。
