# 小说创作项目：《高武：体检最废？我的禁忌天赋无限升级》

> **项目说明**：本目录用于创作小说《高武：体检最废？我的禁忌天赋无限升级》。本文档为该小说的核心设定集与开发需求文档，用于指导创作，并作为 AI 助手生成内容、润色和设定校验的核心上下文参考。

---

## 一、核心基础设定

- **书名**：《高武：体检最废？我的禁忌天赋无限升级》
- **字数目标**：150万字以上长篇男频连载
- **题材/类型**：都市高武 / 校园逆袭 / 禁忌成长流 / 热血爽文
- **一句话简介（Logline）**：
  > 高三体检这天，所有人都认定陆燃是全校最废的垃圾天赋，只有他自己看见了面板上那行猩红小字——吞噬、复制、进化，世间一切天赋，皆可成为他的养料。
- **核心主题/立意**：小人物逆袭、强者秩序的撕裂、被踩进泥里的人如何反过来改写规则。
- **基调与文风**：快节奏、强冲突、强代入、压迫感足、爽点密、章末钩子强。
  - **消除AI味**：禁用总结性结尾、翻译腔、心理分析式描写，对话去标准化
  - **短句为王**：段落不超过4行，多用动作词汇（砸、撞、崩、裂、撕）
  - **感官沉浸**：调动视觉、听觉、嗅觉、触觉，营造身临其境感
  - **接地气细节**：油污、铁锈、汗味、廉价香水、生活化场景

## 二、世界观设定

### 2.1 时代与背景
- **时间**：新历 2068 年
- **地点**：大夏联邦，江北基地市
- **社会结构**：异界裂缝降临后，城市被高墙、军方、武盟和财团共同维系。武道天赋决定阶层，学校体检结果几乎决定学生未来去向。

### 2.2 核心机制
- **力量体系/科技水平**：现代都市框架下融合气血修炼、天赋序列、战技芯片、秘境资源。
- **特殊势力/组织**：
  1. **大夏武盟**：掌握武道认证、秘境开发和强者秩序，是明面上的核心权力机构。
  2. **星渊财团**：垄断药剂、兵器和体检设备，暗中参与高危天赋筛查与禁忌实验。
  3. **黑烬会**：活跃于裂缝区和地下市场，专门猎杀特殊天赋者，追逐禁忌源头。

## 三、角色档案

### 主要角色
1. **陆燃**
   - **身份/职业**：江北三中高三学生，贫民区出身
   - **性格特征**：狠、稳、记仇，外冷内烈，不爱废话
   - **核心动机**：先活下去，再让所有踩过他的人低头，最后查清父亲失踪和自己天赋异常的真相
   - **人物弧光**：被判最废、受尽羞辱的底层少年 -> 撕开天才垄断体系的禁忌强者

2. **楚清月**

   - **核心动机**：摆脱家族安排，用绝对实力决定自己的人生，也想看清陆燃到底藏着什么秘密

## 四、故事大纲

- **第一幕：引子与觉醒**
  - 陆燃在高三统一体检中被判定为最低等废脉，遭全校围观羞辱，连升学名额都要被取消。
  - 当晚他在废弃训练馆被周天雄等人围堵，濒死之际觉醒禁忌天赋“万象熔炉”，第一次吞掉别人天赋碎片完成反杀。

- **第二幕：冲突与挑战**
  - 陆燃借校内排名赛、裂缝试炼、城市选拔赛一路逆袭。
  - 他一边升级，一边发现体检设备对自己做过异常遮蔽，父亲当年的失踪也与禁忌实验有关。
  - 校园层面的敌人很快不够看，真正盯上他的变成世家、财团和黑烬会。

- **第三幕：高潮与结局**
  - 陆燃在禁区深处找到父亲留下的线索，得知自己的天赋不是觉醒，而是被提前埋入体内的“钥匙”。
  - 他必须在吞噬失控与彻底掌控之间作出选择，并在世界级灾变中反过来夺走诸强权柄。
  - 最终走向是主角从被规则判死的废物，成长为亲手重写高武秩序的人。

## 六、创作工具与指南

### 📚 核心文档说明

#### 1. **writer_role.md** - 作者人设与写作铁律
- **定位**：定义顶尖网文作家的写作哲学
- **核心**：消除AI味、短句节奏、感官沉浸、商业元素规范
- **包含**：词汇偏好表、单章结构、人物塑造指南、禁忌清单

#### 2. **chapter_prompt.md** - 章节生成模板  
- **定位**：具体的章节创作指令集
- **核心**：情感弧线、主题关键词、必要元素、写作示例
- **使用**：填写“本章目标”后直接生成正文

#### 3. **characters.md** - 人物档案库
- **定位**：所有角色的深度设定
- **核心**：动机、关系、成长线、行为模式、对话示例
- **包含**：人物关系图谱、好/坏对话示例

#### 4. **timeline.md** - 情节规划图
- **定位**：第一卷详细大纲与长线规划
- **核心**：章节指引、伏笔追踪、名场面设计、情绪目标
- **包含**：伏笔追踪表、关键名场面描述、长期规划

#### 5. **world.md** - 世界观构建
- **定位**：力量体系、势力划分、场景描写指南
- **核心**：具体场景描写示例、势力互动模式、资源描写
- **包含**：力量体系具体表现、稀缺资源描写示例

### 🎯 创作流程建议

#### 步骤1：确定章节位置
- 查看 `timeline.md` 确定本章在整体结构中的位置
- 明确本章的**情绪目标**和**关键功能**

#### 步骤2：填写章节模板  
- 打开 `chapter_prompt.md`
- 填写“当前阶段”、“上一章结尾”
- 设定“主要冲突”、“核心爽点”、“章节钩子”
- 确定“必须出现的人物/道具/信息”

#### 步骤3：参考具体指导
- **人物对话**：参考 `characters.md` 中的对话示例
- **场景描写**：参考 `world.md` 中的描写要点
- **写作风格**：遵循 `writer_role.md` 中的铁律
- **伏笔管理**：查看 `timeline.md` 中的伏笔追踪表

#### 步骤4：生成与校验
- 生成正文后，检查是否：
  - 开场200字内进入冲突
  - 中段不断加压
  - 后段有明确爽点
  - 结尾卡在悬念处
  - 消除AI味（无总结结尾、无翻译腔）

### ⚡ 快速提醒卡片

#### ✅ 必须做到的
- 短句快节奏，段落不超过4行
- 情绪用动作展示，不说“他感到愤怒”
- 对话口语化带火药味
- 结尾戛然而止，不留总结
- 调动多感官（视觉、听觉、嗅觉、触觉）

#### ❌ 绝对禁止的  
- 总结性结尾（“未来的路还很长……”）
- 翻译腔（“宛如”、“不禁”、“值得一提的是”）
- 心理分析式描写（“他感到十分愤怒”）
- 华丽辞藻堆砌
- 冗长复合句（超过25字）

#### 🔥 爽点制造器
- **前期**：憋屈感拉满 → 反杀解气
- **中期**：规则压迫 → 暴力破局  
- **后期**：势力围剿 → 反向吞噬
- **永恒**：具体动作 > 抽象描述

### 📈 质量评估标准

#### A级章节（爆款潜质）
- 读者看完必须点下一章
- 社交平台可截取传播的名场面
- 情绪曲线完整（压抑→对抗→爆发→悬念）

#### B级章节（合格产品）  
- 完成章节基本功能（推进剧情、展示人物）
- 有明确爽点和钩子
- 无明显AI味

#### C级章节（需要重写）
- 节奏拖沓，信息密度低
- AI味明显（总结结尾、翻译腔）
- 爽点模糊，钩子无力

## 🧭 执行层工作流（从多版本到定稿）

### 当前问题
- 历史多版本已完成融合，当前重点从“比稿管理”转为“稳定连载推进”。
- 后续若再开多模型并行，建议临时分支生成，锁稿后立即回收至 `drafts/chapter_XXX.md` 单线版本。

### 定稿流程（建议固定执行）
1. **写作阶段**：在 `drafts/chapter_XXX.md` 维护单线草稿（如需并行，仅做临时稿）。
2. **评审阶段**：在 `drafts/选稿追踪表.md` 记录“需修问题、锁稿状态”。
3. **定稿阶段**：将最终章写入 `published/第XXX章_章节名.md`（只保留一个权威版本）。
4. **回填阶段**：定稿后同步更新 `settings/timeline.md`（状态/伏笔）、`settings/characters.md`（角色状态变化）、`settings/world.md`（新增规则）。

### 命名与状态规范
- 草稿文件：`drafts/chapter_XXX.md`
- 定稿文件：`published/第XXX章_章节名.md`
- 章节状态仅允许四种：`draft` / `reviewing` / `locked` / `published`
  - `draft`：草稿生成中
  - `reviewing`：正在比稿融合
  - `locked`：内容锁定，等待入库
  - `published`：已进入 `published/`，作为唯一后续依据
