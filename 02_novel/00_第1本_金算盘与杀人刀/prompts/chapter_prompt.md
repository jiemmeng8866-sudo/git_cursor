# 章节生成模板

## 已知设定

- **书名**：《玉算谋国：朱门半夏》
- **题材**：古代权谋 × 商业经营 × 女性成长 × 高智商言情
- **女主**：沈半夏（商贾之女/落魄千金，极度清醒，信奉真金白银比人心更可靠）
- **男主**：萧祈（权倾朝野的异姓王/隐忍复仇者，腹黑多疑，杀伐果断）
- **男二**：裴正（大理寺卿，冷酷无情，嗅觉敏锐）

## 章节指令模板

```
## 章节信息
- 卷数：[当前卷]
- 章节号：[第X章]
- 字数目标：[2800-3200字]

## 核心目标
[用一句话说明本章要达成什么效果]

## 情感弧线
- 压抑：[前20%的困境或问题]
- 对抗：[中间50%的博弈和拉扯]
- 爆发：[后20%的高潮或反转]
- 悬念：[最后10%的钩子]

## 必要元素

### 必须出现的人物
- [角色A]：[出场目的]
- [角色B]：[出场目的]

### 必须涉及的道具/信息
- [道具/信息]

### 伏笔操作
- 埋设：[新伏笔]
- 回收：[旧伏笔]

## 写作铁律

1. 禁用现代词汇（商业模式、供应链等）
2. 禁用心理分析式描写
3. 场景要有烟火气（嗅觉、触觉、听觉）
4. 短句快节奏，段落不超过4行
5. 结尾断在悬念处
```

## 写作示例

### 商战类写法

**✅ 好写法（动作驱动，信息密度高）**
"沈半夏把五枚铜钱拍在柜台上，指节泛白。驿卒眼皮跳了跳，没接。她盯着他，声音哑得像吞了炭：'外头那草棚要塌了。你现在不拿，待会儿可就不是这个价了。'"

**❌ 坏写法（情绪描述，信息量低）**
"沈半夏感到非常绝望，但她还是鼓起勇气把铜钱放在柜台上。驿卒看起来很惊讶，似乎没想到她还有钱。沈半夏心想，如果她不争取一下，就真的没有活路了。"

### 权谋类写法

**✅ 好写法（话里有话，留白）**
"裴正把卷宗推过来，不说话。沈半夏没接。她把算盘搁了上去，指尖在木框上轻轻一叩。'大人给我看的是账，我给大人看的，也是账。'"

**❌ 坏写法（直接揭露）**
"裴正拿出了沈家做假账的证据，沈半夏很紧张，但她还是冷静地拿出了复式记账法的证据。她解释说二房在账目上做了手脚。裴正听完后明白了真相。"

### 动作类写法

**✅ 好写法（短句、节奏感）**
"刀来了。沈半夏没躲。她算准了那一刀不会落在自己身上。萧祈的铁刀在半空中截住了另一把刀。火星溅在她脸上，烫出个泡。"

**❌ 坏写法（拖沓、解释过多）**
"就在这时，一把刀突然向沈半夏砍来，她非常害怕，但她知道萧祈一定会救她。果然，萧祈及时出手，用他的黑铁直刀挡住了那把刀。"

## 不同章节类型的侧重点

| 类型 | 侧重点 | 示例章节 |
|------|--------|----------|
| **商战章** | 数字、合同、算账、谈判 | 第2章买断路权、第35章做空侯府 |
| **权谋章** | 对话机锋、朝堂对峙、证据战 | 第10章朝堂三审、第34章与虎谋皮 |
| **动作章** | 短句、感官冲击、危机感 | 第33章疯狗见血、第1章草棚反杀 |
| **过渡章** | 信息铺垫、人物塑造、伏笔埋设 | 第14章转战江南、第20章藏身吴州 |

## 绝对禁止

- ❌ 总结性结尾（"这一夜，沈半夏明白了一个道理……"）
- ❌ 翻译腔（"如果……的话"、"当……的时候"）
- ❌ 心理分析式描写（"她意识到……"、"她感到……"）
- ❌ 机械式对话（"你好"、"再见"、"谢谢"）

## 质量评估

| 等级 | 标准 |
|------|------|
| A级 | 节奏紧凑，爽点密集，结尾悬念拉满 |
| B级 | 故事推进正常，但缺乏亮点 |
| C级 | 节奏拖沓，AI味重，需要重写 |
