# 专业术语介绍

| 术语     | 介绍                                                         |
| -------- | ------------------------------------------------------------ |
| 决策引擎 | 作为风控模型的载体：根据复杂的业务逻辑抽象化剥离出来的业务规则，进而命中不同分支组合层层规则递进运算输出决策结果的产品；其中业务规则分为普通规则、循环规则和优先级规则，每条规则均由变量、表达式、条件值、决策结果组成，因此引擎存在入参和出参的明细表； |
| 模型     | 风控引擎中使用的模型多为数据模型，即引擎仅提供入参变量的配置，调用已封装好的模型，模型输出返回对应的结果作为引擎的出参变量； |
| 出入参   | 用于决策引擎调用模型的数据（变量）即入参，模型输出结果返回给引擎，引擎输出给后台的结果即出参； |
| 沙箱     | 为我行人员提供逻辑隔离的运行环境，用于开发、测试、不受信任的代码或应用程序，是一种非生产业务用途的工具；主要用于模型上线前的数据逻辑检验，为了保证模型在分析环境和生产环境的结果一致性；其可由风险模型联机使用原始报文数据作为模型入参测试，缩短开发周期，提高模型时效性的封闭性测试环境； |
