| 类别                      | 专业术语                                                     | 简要介绍                                                     |
| ------------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 计算口径                  | LUM                                                          | 微众银行负责持续管理的贷款                                   |
| 表内                      | 未进行出表操作的我行份额贷款资产                             |                                                              |
| 分流                      | 小微银银2.0和小微银银2.1的联合贷划分为我行自营额度的部分贷款 |                                                              |
| 出表资产                  | [ABS](https://ABS-ABN.md)                                    | 一种出表操作：资产支持证券（Asset-Backed Securities）        |
| [ABN](https://ABS-ABN.md) | 一种出表操作：非金融企业资产支持票据 (Asset-Backed Medium-term Notes) |                                                              |
| 不良资产转让              | 一系列降低和分散银行不良资产风险的处理方案                   |                                                              |
| 代偿                      | 保险代偿                                                     | 一种违约由保险公司承担债务，代为偿还的代偿方式               |

| 特殊套餐                  | 锁期                                                         | 一种由客户选定贷款方案，且在规定期限内不能提前还款的贷款套餐 |
| 万一贷                    | 我行一种可配置万一、万二等多种利率的特殊贷款套餐             |                                                              |
| 纾困                      | 银行由于借款企业财务状况恶化，或无力还款而对借款合同还款条款作出调整的贷款方案 |                                                              |
| 其他                      | 主标尺                                                       | 我行针对不同模型分数不可比性等问题所构建的统一风险测量体系   |
| 其他                      | 决策引擎、沙箱、模型、出入参等                               |                                                              |
