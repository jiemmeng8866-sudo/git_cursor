# 金额不良率

## 1 业务含义

***金额不良率（不良贷款率）：***不良贷款占总贷款余额的比例。不良贷款是指在评估银行贷款质量时，将贷款按风险基础分为正常、关注、次级、可疑和损失五类，即，该指标即次级类、可疑类、损失类的贷款（为不良贷款）占正常在贷的总余额的比例（五级分类字段classifyresult in ('03','04','05')）；

***客户不良率：***五级分类为次级、可疑、损失三类的客户占正常在贷的总户数的占比（classifyresult in ('03','04','05')）；

*不良贷款率是评价银行信贷资产安全状况的重要指标。不良贷款率高，说明回收贷款的风险大，反之说明收回贷款的风险小，该指标常用于直观评估贷款整体放款的风险水平和客户质量，但其不能详细反映真实的坏账风险，存在高估/低估的风险，需结合时点和拆分维度来观察不良率的波动性；*

**五级分类：**正常类、关注类、次级类、可疑类、损失类；具体可参考**不良资产处置文档；**

**参考资料：****[We学堂-风险管理常用指标分享；](https://learning.weoa.com/bbs/#/postdetail/dfd1f1ed-29cc-4e50-8fd8-62df5776cee3)**

## 2 贷款余额-计算口径

***原值口径：***balance原始字段

***估值口径：***原始贷款本金会迭代乘以随着期数变化的折扣率，具体计算逻辑需联系风险管理部的elliewu(吴娴)；表内借据的估值余额可通过brm_wyd_ods_mask.rrs_bdap_ln_lncf_qyjr_wzyd表直接取（请注意计算的产品范围！！！）

***LUM（管理贷款）口径：***保险代偿、担保代偿、视同核销、零元回购均需纳入余额统计范围

```Plain
case when coalesce(t.is_lyhg,0) = 1 then 0                     --剔除零元回购
     when coalesce(t.is_sthx,0) = 1 then 0                     --剔除视同回购    
     when coalesce(t.is_bxdc,0)=1 then we_balance_bxdc         --剔除保险代偿    
     when coalesce(t.is_dbdc,0)=1 then we_balance_dbdc         --剔除担保代偿    
     else balance end as lum_balance 
```

***表内口径：***保险代偿、担保代偿、视同核销、零元回购、小微银银2.1均需纳入余额统计范围

```Plain
case when coalesce(is_outside_table_package,0) = 0 and source_tbl in ('cfcm', 'mcfcm','ymcfcm') 
    then (
        case when coalesce(t.is_lyhg,0) = 1 then 0                --剔除零元回购
        when coalesce(t.is_sthx,0) = 1 then 0                     --剔除视同回购           
        when coalesce(t.is_xwyy_21,0)=1 then we_balance_xwyy_21   --剔除小微银银2.1           
        when coalesce(t.is_bxdc,0)=1 then we_balance_bxdc         --剔除保险代偿          
        when coalesce(t.is_dbdc,0)=1 then we_balance_dbdc         --剔除担保代偿           
        else balance end
    )
    else 0 end as bn_balance        
```

## 3 参考代码

```SQL
--计算核销前/后+LUM口径/表内口径+时点-不良贷款率
select
     sum(case when duestatus in ('0','1') and classifyresult in ('03','04','05') then balance else 0 end)/sum(case when duestatus in ('0','1') then balance else 0 end) as hxh_balance_yz                             --原值口径 核销后金额不良率
    ,sum(case when duestatus in ('0','1') and classifyresult in ('03','04','05') then balance_lum else 0 end)/sum(case when duestatus in ('0','1') then balance_lum else 0 end) as hxh_balance_lum                    --LUM口径 核销后金额不良率
    ,sum(case when duestatus in ('0','1') and classifyresult in ('03','04','05') then balance_bn else 0 end)/sum(case when duestatus in ('0','1') then balance_bn else 0 end) as hxh_balance_bn                         --表内口径 核销后金额不良率
from(--step1：计算LUM口径和表内口径的不良资产
    select *
    --表内口径：保险代偿、担保代偿、视同核销、零元回购、小微银银2.1均需纳入余额统计范围
    ,case when source_tbl in ('cfcm', 'mcfcm','ymcfcm') and coalesce(is_outside_table_package,0) = 0
    then (case when coalesce(is_lyhg,0) = 1 then 0                    --剔除零元回购
               when coalesce(is_sthx,0) = 1 then 0                    --剔除视同核销
               when coalesce(is_xwyy_21,0)=1 then we_balance_xwyy_21  --剔除小微银银2.1
               when coalesce(is_bxdc,0)=1 then we_balance_bxdc        --剔除保险代偿
               when coalesce(is_dbdc,0)=1 then we_balance_dbdc        --剔除担保代偿
               else balance end) else 0 end as balance_bn
    --LUM（管理贷款）口径：保险代偿、担保代偿、视同核销、零元回购均需纳入余额统计范围                                    
    ,case when coalesce(is_lyhg,0) = 1 then 0                          --剔除零元回购
          when coalesce(is_sthx,0) = 1 then 0                          --剔除视同核销
          when coalesce(is_bxdc,0)=1 then we_balance_bxdc              --剔除保险代偿
          when coalesce(is_dbdc,0)=1 then we_balance_dbdc              --剔除担保代偿
          else balance end as balance_lum                              --LUM口径
    from brm_ent_dm_work.dwd_mcfcm_business_duebill_etl_ds
    where duestatus in ('0','1','2','3') 
    and ds = '${run_date}'
    and is_v4_l1_wyd_kcd = 1 --限定企金整体
)t
```
